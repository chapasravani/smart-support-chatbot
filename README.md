# 🤖 Smart Support Chatbot

A simple and interactive AI-based customer support chatbot built using Flask, HTML, CSS, and JavaScript.

---

## 🚀 Features

- 💬 Handles common customer queries  
- 📦 Order tracking responses  
- 💰 Refund & cancellation support  
- 🔐 Password reset guidance  
- 🎓 Study, food & travel FAQs  
- 🤖 Friendly conversational replies  

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- Bootstrap (UI styling)

## ▶️ How to Run

1️⃣ Clone repo  
```
git clone https://github.com/chapasravani/smart-support-chatbot.git
```

2️⃣ Install dependencies  

```
pip install flask
```

3️⃣ Run project  

```
python app.py
```

4️⃣ Open browser  

```
http://127.0.0.1:5000
```
